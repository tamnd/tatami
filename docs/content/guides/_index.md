Each guide is built around a job rather than an API surface: writing typed columns
and reading them back, bringing existing Parquet output into the format, stitching
many files into one collection, and turning a corpus into a keyword search index
that answers in microseconds. They assume you have worked through the
[quick start](/getting-started/quick-start/).
